# Лабораторна робота №1 — Farm Store

## Тема
Розробка елементарних програм на мові програмування C#.

## Проєкт
Консольна програма **Farm Store** рахує вартість покупки у фермерському магазині.

## Що використано
- введення даних через `Console.ReadLine()`;
- виведення через `Console.WriteLine()`;
- конвертація `Convert.ToDouble()`;
- математичні методи `Math.Round()`, `Math.Sqrt()`, `Math.Pow()`;
- випадкова знижка через `Random().NextDouble()`;
- кольори консолі через `Console.ForegroundColor`.

## Як запустити
1. Відкрити папку `FarmStore` у Visual Studio або VS Code.
2. Запустити команду:

```bash
dotnet run
```

Або відкрити файл `FarmStore.sln` у Visual Studio і натиснути **Start**.
